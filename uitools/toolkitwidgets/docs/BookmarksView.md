# BookmarksView - Qt Widgets UI control

The [BookmarksView - Qt Widgets UI control](https://developers.arcgis.com/qt/toolkit/api-reference/esri-arcgisruntime-toolkit-bookmarksview.html) displays a collection of viewpoints represented by bookmarks from either Webmap/Webscene or are programmatically defined. When the user selects a bookmark from the provided list, the viewpoint in the geoView is set to the new bookmark's view extent.
